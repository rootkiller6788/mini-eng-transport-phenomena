# Course Tree: Prerequisites

|- Linear Algebra (matrix ops, solvers)
|  |- Numerical Methods (iterative, convergence)
|- Fluid Mechanics (Navier-Stokes)
|  |- Conservation Laws
|  |- Boundary Layer Theory
|- Heat Transfer (conduction, convection)
|  |- Dimensionless Groups
|- Mass Transfer (diffusion, convection)
|- Numerical Analysis
|  |- FDM, FVM
|  |- Stability (CFL, von Neumann)
|  |- Error Analysis
|- Programming (C, data structures)
|- Turbulence Modeling
   |- RANS, k-epsilon, k-omega SST

Downstream dependencies:
|- Advanced CFD (LES, DNS, multiphase)
|- Compressible Flow CFD
|- Combustion CFD
|- Multiphysics Coupling
|- Optimization (adjoint)
